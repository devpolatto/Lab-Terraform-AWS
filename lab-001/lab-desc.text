Desenvolver uma infraestrutura da seguinte forma:

1. Criar duas Ec2
2. Um servidor web(Nginx) em uma das Ec2
3. Um servidor DNS em uma das Ec2

OBS: 
     - Cada instancia deve estar com IP fixo;
     - Apenas a Ec2(Nginx) deve recer um ELP
     - Regitra um domonio com polatto.tec e apeontar para o servidor NGINX
